# Online chat/UI sound sources to review

These sources are useful when building online Echo-Chat sound packs. Always check the license on the exact sound you download or link.

## JavaScript-first options

- **simple-notification-sounds** — MIT JavaScript library with browser UMD CDN file. Echo-Chat has an automatic bridge for this URL:
  `https://cdn.jsdelivr.net/npm/simple-notification-sounds@1.0.0/dist/simple-notification-sounds.umd.js`
- **soundcn** — open-source UI sound registry with many TypeScript modules containing inline base64 data URIs. These can be adapted into Echo-Chat `.js` sound packs.

## Audio-library options for custom packs

- **SOUNDICONS UI/UX sounds** — includes CHAT, MAIL, PHONE, SELECT, OPEN, CANCEL, and related UI categories.
- **SND.dev** — curated UI sound kits for product interfaces.
- **Pixabay Sound Effects** — searchable UI and notification effects.
- **Mixkit Notification Sound Effects** — short notification sounds like message pop, bell notification, sci-fi click, interface start/back, and positive/error tones.
- **Zapsplat Alerts and Prompts** — broad alert/prompt/UI effects; review account/license requirements before using in a bundled product.

## Practical mapping for Echo-Chat

- **Private message**: chat pop, soft ping, bubble, short attention.
- **Room message**: quiet chime or soft tick; avoid loud alerts.
- **Group message**: slightly different ping from DM.
- **Room invite / friend request**: bell, doorbell, rising success tone.
- **Join / leave**: page flip, soft door/open/close cue.
- **File / torrent**: digital drop, transfer tick, completion chime.
- **Error / warning**: low buzz, warning pulse, error pop.

## Admin Panel shortcut buttons

Admins can open this source list directly from **Admin Panel → System → Notification sounds → Open sound source list**. Use these sources as places to review sound ideas; do not assume every file is free for redistribution. Check the license for the exact file you download or link.
